🎨 AI Code Reviewer — Frontend

🧠 Overview

This frontend is a modern web interface for the AI Code Reviewer system.

Users can:

- Write Python code
- Analyze it in real-time
- View issues, score, and suggestions
- Get improved code from AI

---

⚙️ Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Monaco Editor

---

🧩 Features

🧑‍💻 Code Editor

- Monaco Editor
- Python syntax highlighting
- Real coding experience

---

🚀 Analyze Flow

- Sends code → backend
- Displays loading state
- Handles errors gracefully

---

📊 Score System

- Dynamic score (0–100)

Score Range| Color
> 80| Green
50–80| Yellow
< 50| Red

---

⚠️ Issues Panel

- Displays detected issues
- Clean structured UI

---

💡 Suggestions Panel

- AI-generated suggestions
- Easy to read format

---

🧠 Improved Code

- Syntax-highlighted output
- Copy button

---

🧪 Test Case System (🔥 Key Feature)

- Side dropdown panel
- 10 predefined test cases
- “Use” → loads into editor
- “Copy” → clipboard

---

🔗 API Integration

POST /analyze

---

▶️ Run Frontend

Commands:
npm install
npm run dev
